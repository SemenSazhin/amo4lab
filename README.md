# Калькулятор для Code Review - amo4lab

### Установка

```bash
git clone https://github.com/SemenSazhin/amo4lab.git
cd amo4lab
```

## Задачи для улучшения (что можно исправить в code review)

### 🐛 Баги для исправления:
1. Деление на ноль ломает дальнейшие вычисления
2. Нет обработки очень длинных чисел
3. Калькулятор не работает с клавиатуры
4. При нажатии "=" несколько раз подряд происходит ошибка

### ✨ Функции для добавления:
1. Кнопка "%" (проценты) не подключена к интерфейсу
2. Нужна кнопка смены знака (+/-)
3. История вычислений
4. Поддержка клавиатуры

### 🎨 Улучшения дизайна:
1. Адаптивность для мобильных устройств
2. Темная/светлая тема
3. Анимации нажатий

### 🔧 Рефакторинг:
1. Убрать глобальные переменные
2. Удалить неиспользуемую функцию
3. Добавить JSDoc комментарии
4. Вынести логику в отдельные модули

## Как работать над заданием

1. **Создайте свою ветку**: `git checkout -b feature/ваше-имя`
2. **Внесите изменения** (выберите любую задачу из списка выше)
3. **Создайте Pull Request** в ветку `main`
4. **Проведите code review** коллегам
5. **Исправьте замечания** и получите апрув
6. **Владелец репозитория** мержит PR

## Что проверять на code review

- ✅ Качество кода
- ✅ Наличие обработки ошибок
- ✅ Производительность
- ✅ Безопасность
- ✅ Читаемость кода
- ✅ Следование best practices
